# ClientFlow

ClientFlow é um sistema de gerenciamento de clientes em desenvolvimento, criado como projeto de portfólio para fortalecer minha jornada na área de programação.

## Sobre o projeto

A proposta do ClientFlow é simular uma plataforma moderna de cadastro e acompanhamento de clientes, com foco em organização, interface profissional e evolução gradual para uma aplicação full stack.

No estado atual, o projeto já possui uma interface premium inspirada em plataformas modernas, além de funcionalidades reais no frontend.

## Status atual

Projeto em desenvolvimento.

Atualmente o frontend já conta com:

- cadastro de clientes
- listagem de clientes
- edição de clientes
- exclusão de clientes
- contadores de status
- persistência com localStorage
- componentização com React
- layout responsivo
- visual premium com identidade mais forte

## Tecnologias utilizadas

### Frontend

### Backend
- Node.js
